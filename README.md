###  统一任务调度平台 for golang

###### 1、pre
* install golang env

  https://github.com/golang/go
  
#### 2、install  
  
 

######    get beego 
*        go get github.com/astaxie/beego
#####    get scheduler
*        go get github.com/shotdog/scheduler
#####   get quartz
*       go get github.com/shotdog/quartz
##### get mysql driver for golang
*       go get  github.com/go-sql-driver/mysql


#### run

*       cd $GOPATH
*       cd src
*       cd scheduler
*       go build main.go
*       ./main
